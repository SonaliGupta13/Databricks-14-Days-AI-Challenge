# Databricks-14-Days-AI-Challenge
Welcome to my submission repository for the **Databricks 14 Days AI Challenge** hosted by Indian Data Club, sponsored by Databricks, and organized by Codebasics & Indian Data Club.

This challenge is designed to build strong foundations in **Databricks, Apache Spark, PySpark, Delta Lake**, and **AI-powered analytics** through structured daily learning and hands-on practice.

### About the Challenge
The **Databricks 14 Days AI Challenge** focuses on practical, real-world data engineering and analytics workflows using Databricks Lakehouse architecture.

By the end of this challenge, we gain exposure to:

- Modern data engineering best practices
- Scalable analytics with Spark
- Lakehouse & Medallion Architecture
- Governance, orchestration, and performance tuning
- ML & AI workflows with Databricks

###  Challenge Overview

- **Duration:** 14 Days (Learning) + 7 Days (Project Phase)
  
- **Platform:** Databricks Community Edition

- **Dataset:** E-commerce Behavior Dataset (Kaggle)

### Focus Areas
- Databricks Platform Fundamentals
- Apache Spark & PySpark
- Delta Lake & ACID Transactions
- Medallion Architecture (Bronze–Silver–Gold)
- Workflows & Job Orchestration
- Unity Catalog & Data Governance
- SQL Analytics & Performance Optimization
- MLflow & AI-powered Analytics

###  Dataset Details
**Dataset:** E-commerce Behavior Data from Multi-category Store

**Source:** Kaggle (Provided by REES46 Open CDP)

##### Files Used:
- 2019-Oct.csv
- 2019-Nov.csv

##### Key Columns:

event_time, 
event_type (view, cart, purchase), 
product_id, 
category_code, 
brand, 
price, 
user_id, 
user_session
